What is a folder on your computer (OS-level)?

What is a package in Java (language-level)?
Package is a logical grouping of classes and interfaces in Java.

Why do packages exist?
Packages help to organize code and avoid conflicts.

What problems do packages solve that folders alone cannot?
Folders just store files and packages organize the structure of code.

Cause if the folder structure wouldn't matсh the package name, copmpiler won't be able to find files.

What kind of error or conceptual issue happens when they don’t match?
